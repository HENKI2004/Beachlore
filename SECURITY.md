# Security Policy

## Security Status

The iso26262-ecc-analyzer is provided as a research tool on a best-effort basis and has not undergone formal security auditing.

## Reporting a Security Concern or Conduct Issue

We take the security and community health of the **iso26262-ecc-analyzer** project seriously. 

To report a security vulnerability or a violation of the Code of Conduct, please use GitHub's **Private Vulnerability Reporting** feature. This ensures your report is handled privately and securely without exposing your contact information or the details of the report to the public.

### How to report:
1. Navigate to the [Security tab](https://github.com/HENKI2004/iso26262-ecc-analyzer/security) of this repository.
2. Click on **Vulnerability reporting** in the left sidebar.
3. Click the **Report a vulnerability** button to open a private report.

We will acknowledge receipt of your report and get in contacted with you.
